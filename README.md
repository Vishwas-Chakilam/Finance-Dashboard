# FinTrack — Finance Dashboard Demo

FinTrack is a frontend finance dashboard built by **Vishwas Chakilam** as part of a frontend development assignment.  
It focuses on clean UI, realistic product flows, and modern React patterns rather than backend complexity.

## Tech Stack

- **React 18 + TypeScript**
- **Vite** for bundling and dev server
- **React Router** for routing & protected routes
- **@tanstack/react-query** for async state patterns
- **Tailwind CSS + shadcn/ui** for design system components
- **Recharts** for charts
- **Vitest + Testing Library** for tests

## Features

- Landing page with marketing-style copy and navigation
- Mock authentication (email + password) and logout
- Protected dashboard routes (dashboard, transactions, analytics, budgets, accounts, settings)
- Responsive charts and summary cards
- Transactions table with filters
- Budgets and accounts overview
- Settings page with profile, preferences, notifications, and data tools
- Global toasts, transitions, and custom scrollbars

## Local Development

```bash
npm install
npm run dev
```

Run tests:

```bash
npm test
```

Run lint:

```bash
npm run lint
```
